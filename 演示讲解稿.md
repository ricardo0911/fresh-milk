# 鲜奶订购系统 - 项目演示讲解稿

---

## 一、项目概述与技术路线

**项目名称**：Fresh Milk 鲜奶订购小程序  
**核心定位**：为用户提供鲜奶产品浏览、在线订购、周期订阅、会员服务等功能的完整电商生态

**技术栈**：
- **后端**：Django 4.x + Django REST Framework + MySQL 8.0
- **管理端**：Vue 3 + Vite + Element Plus
- **用户端**：微信小程序原生开发
- **数据存储**：MySQL 数据库 + 本地文件存储

---

## 二、系统角色划分

### 角色 1：管理员（管理端）
**入口**：Web 管理后台（Vue 单页应用）

**核心功能**：
- **仪表盘**：查看系统统计数据（用户总数、产品总数、订单总数、总收入、今日数据）
- **待处理事项**：待发货订单、待处理反馈、库存预警
- **用户管理**：查看注册用户列表、搜索用户、管理用户状态、调整积分
- **商品管理**：
  - 添加/编辑商品信息
  - 上传商品图片
  - 设置分类、价格、库存
  - 设置周期购属性
- **订单管理**：查看所有订单、处理发货、处理退款
- **优惠券管理**：创建优惠券、设置规则、发放优惠券
- **周期购管理**：查看订阅计划、管理配送
- **评论管理**：查看用户评论、删除不当评论
- **反馈管理**：查看用户反馈建议、回复反馈、标记状态

### 角色 2：普通用户（小程序端）
**入口**：微信小程序

**核心功能模块**：
1. **首页/发现**
   - 轮播图推荐
   - 周期购快捷入口
   - 猜你喜欢商品推荐
   - 热门推荐、新品上市

2. **商品浏览**
   - 分类浏览（常温奶、低温奶、酸奶等）
   - 商品详情（规格、价格、库存）
   - 商品搜索
   - 加入购物车

3. **购物车**
   - 添加/删除商品
   - 修改数量
   - 选择结算商品

4. **订单管理**
   - 提交订单（选择地址、优惠券）
   - 微信支付
   - 查看订单状态（待付款、待发货、待收货、待评价）
   - 物流追踪
   - 申请退款/售后

5. **周期购订阅**
   - 创建周期配送计划
   - 选择配送产品
   - 管理订阅计划（暂停/恢复/取消）
   - 享受订阅专属折扣（9折）

6. **会员体系**
   - 查看会员等级（普通/银卡/金卡/铂金）
   - 会员折扣权益
   - 积分明细
   - VIP 会员购买

7. **优惠券**
   - 领取优惠券
   - 积分兑换优惠券
   - 下单时使用优惠券

8. **个人中心**
   - 用户信息展示
   - 我的订单（各状态入口）
   - 收货地址管理
   - 我的评价
   - 意见反馈
   - 联系客服
   - 设置、退出登录

## 三、核心业务流程详解

### 业务流程 1：用户注册与登录

**完整流程**：
1. **进入小程序**：
   - 用户打开微信小程序
   - 进入首页可浏览商品

2. **触发登录**：
   - 点击"我的"页面
   - 提示登录/注册

3. **注册**：
   - 输入手机号
   - 设置密码
   - 完成注册
   - 系统自动创建用户档案，初始为普通会员

4. **登录**：
   - 输入手机号和密码
   - 验证成功后进入小程序
   - 获取用户信息和会员权益

5. **会员权益生效**：
   - 系统自动计算会员等级
   - 享受对应折扣
   - 积分账户初始化

---

### 业务流程 2：商品浏览与下单购买

**完整流程**：
1. **首页浏览**：
   - 用户进入小程序首页
   - 查看轮播图推荐
   - 查看"猜你喜欢"、"热门推荐"、"新品上市"

2. **商品筛选**：
   - 点击分类进入商品列表
   - 使用搜索框搜索商品
   - 筛选浏览

3. **商品详情**：
   - 查看商品详细信息
   - 查看规格、价格
   - 查看用户评价

4. **加入购物车**：
   - 选择商品
   - 点击"加入购物车"或直接点击"+"按钮
   - 可继续购物

5. **购物车结算**：
   - 进入购物车页面
   - 选择需要购买的商品
   - 点击"去结算"

6. **提交订单**：
   - 选择收货地址
   - 选择优惠券
   - 系统自动计算会员折扣
   - 确认订单金额
   - 点击"提交订单"

7. **支付**：
   - 跳转微信支付
   - 完成支付
   - 订单状态变为"已支付/待发货"

8. **等待配送**：
   - 管理员处理发货
   - 用户可查看物流信息
   - 配送完成后确认收货

9. **评价**：
   - 订单完成后可评价商品
   - 写评语、评分
   - 评价获得积分奖励

---

### 业务流程 3：周期购订阅流程

**使用场景**：用户需要定期配送鲜奶（如每天早上配送）

**完整流程**：
1. **进入周期购**：
   - 首页点击"周期购入口"
   - 或在个人中心点击"周期购服务"

2. **选择商品**：
   - 浏览支持周期购的商品
   - 选择需要订阅的产品

3. **设置订阅计划**：
   - 选择配送频率（每天/每周/每两周/每月）
   - 设置首次配送日期
   - 选择配送数量

4. **订阅优惠**：
   - 系统自动应用周期购折扣（9折）
   - 叠加会员折扣
   - 显示订阅总价

5. **支付订阅**：
   - 确认订阅计划
   - 微信支付
   - 订阅创建成功

6. **管理订阅**：
   - 在"我的"-"周期购"查看所有订阅
   - 可暂停订阅
   - 可恢复订阅
   - 可取消订阅

7. **自动配送**：
   - 系统按设定频率生成订单
   - 按地址配送
   - 用户收到鲜奶

---

### 业务流程 4：优惠券使用流程

**完整流程**：
1. **领取优惠券**：
   - 进入优惠券页面
   - 查看可领取的优惠券
   - 点击领取

2. **积分兑换**：
   - 使用积分兑换优惠券
   - 查看兑换记录

3. **使用优惠券**：
   - 下单时进入优惠券选择页
   - 系统自动筛选可用优惠券
   - 选择合适的优惠券
   - 订单金额自动减免

4. **优惠券类型**：
   - 满减券：满X元减Y元
   - 折扣券：如9折、95折
   - 新用户专享券

---

### 业务流程 5：会员积分体系

**完整流程**：
1. **会员等级**：
   - 普通会员：无折扣
   - 银卡会员：95折（累计消费满一定金额）
   - 金卡会员：9折
   - 铂金会员：85折

2. **积分获取**：
   - 下单消费：消费金额对应积分
   - 评价商品：积分奖励
   - 日常活动：签到等

3. **积分使用**：
   - 积分兑换优惠券
   - 查看积分明细

4. **VIP 会员购买**：
   - 进入 VIP 页面
   - 查看 VIP 权益
   - 购买 VIP 会员（月度/季度/年度）
   - 享受额外 VIP 特权折扣

---

### 业务流程 6：售后与退款流程

**完整流程**：
1. **申请售后**：
   - 用户在订单详情点击"申请退款"
   - 选择退款原因
   - 提交退款申请

2. **管理员处理**：
   - 管理员在后台查看退款申请
   - 审核退款原因
   - 同意/拒绝退款

3. **退款处理**：
   - 同意退款后，原路退回
   - 订单状态更新为"已退款"

4. **用户查看**：
   - 用户在售后页面查看处理进度
   - 查看退款金额到账情况

---

### 业务流程 7：反馈处理流程

**完整流程**：
1. **用户提交**：
   - 用户在小程序"我的"页面点击"意见反馈"
   - 填写反馈内容（文字描述）
   - 选择反馈类型（功能建议/产品质量/配送问题/其他）
   - 提交到系统

2. **管理员处理**：
   - 管理端"反馈管理"查看所有反馈
   - 查看反馈详情（内容、用户、时间）
   - 回复处理意见
   - 标记处理状态（待处理/处理中/已解决）

3. **用户查看**：
   - 用户可在反馈列表查看管理员回复
   - 了解问题处理进度

## 四、系统交互方式

### 1. 数据流转
- **用户操作** → 微信小程序 → Django REST API → MySQL 数据库
- **管理员操作** → Vue 管理后台 → Django REST API → MySQL 数据库
- **商品图片** → 存储在服务器 media 目录 → 通过 HTTP 访问

### 2. 权限控制
- **管理员**：JWT Token 认证，拥有所有管理权限
- **用户**：JWT Token 认证，只能操作自己的数据（我的订单、我的地址、我的收藏）

### 3. 实时同步
- 商品库存变更实时生效
- 订单状态变更用户端立即看到
- 优惠券状态实时更新

## 五、项目亮点

1. **完整的电商生态**：从商品管理到用户下单支付的全链路
2. **周期购订阅**：支持定期自动配送，适合鲜奶类周期性消费商品
3. **会员积分体系**：四级会员等级+积分系统，增强用户粘性
4. **优惠券营销**：多种优惠券类型+积分兑换，促进消费
5. **完整售后流程**：支持退款申请和处理
6. **数据统计**：完整的销售统计、用户分析、热门产品排行
